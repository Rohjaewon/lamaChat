### 설명

React + socket.io를 이용한 실시간 채팅 서비스를 구현합니다.

### 1.시작하기

#### client

`npm run start`

#### server(local)

`node index.js`

### 2.목표

#### basic

1. socket.io를 이용한 실시간 채팅 구현 (clear)
2. Material UI를 이용한 전체 디자인

#### Advanced

3. 소셜 로그인 구현
4. firebase를 이용한 웹서버 생성
5. 배포
